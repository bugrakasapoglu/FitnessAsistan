### Kurulum Adımları

1.  Projeyi bilgisayarınıza indirin:

git  clone [https://github.com/bugrakasapoglu/FitnessAsistan.git]

2.  Proje klasörüne gidin:

cd [FitnessAsistan]

3.  Gerekli paketleri yükleyin:

flutter  pub  get

4.  Uygulamayı başlatın:

flutter  run
